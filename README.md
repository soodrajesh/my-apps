# My Apps Dashboard

A personal portfolio/dashboard page providing public access to all personal apps — no Vercel login required.

## Live URL

Deploy to Vercel and set a custom project name (e.g. `soodrajesh`) to get `soodrajesh.vercel.app`.

## Tech Stack

- Pure HTML/CSS/Vanilla JS — no build step, no dependencies
- Google Fonts (Syne + DM Sans)
- Deployed via Vercel (static hosting)

## Deploy

1. Push this folder to a GitHub repository
2. Import the repo in Vercel
3. Vercel auto-detects static HTML — no build command needed
4. Set the project name to get your preferred URL

## Local Preview

Open `index.html` directly in a browser — no server required.
